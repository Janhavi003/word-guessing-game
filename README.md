# Word Guessing Game

A beginner-friendly full-stack word guessing game built using Python and JavaScript.

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## How to Run

### Backend
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cd backend
python app.py
cd frontend
python -m http.server
